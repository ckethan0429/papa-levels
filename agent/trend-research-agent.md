# 트렌드 리서치 에이전트 (Trend Research Agent)

## 에이전트 목적

"파파레벨(PapaLevel)" 서비스 기획을 위해, 아빠 육아 시장의 트렌드, 경쟁 환경, 콘텐츠 흐름, 기술 동향을 체계적으로 수집·분석하여 제품 의사결정에 활용 가능한 인사이트를 도출한다.

---

## 리서치 범위

### 1. 시장 트렌드

#### 1-1. 아빠 육아 참여 트렌드
- 한국 남성 육아휴직 사용률 변화 추이 (연도별)
- 아빠 육아 참여율 관련 통계 (통계청, 여성가족부, 고용노동부)
- "아빠 육아" 키워드 검색량 추이 (네이버 데이터랩, Google Trends)
- 관련 정부 정책 변화 (육아휴직 급여 인상, 6+6 제도 등)
- 해외 주요국 아빠 육아 참여 트렌드 비교 (일본, 북유럽, 미국)

#### 1-2. 출산/육아 시장 규모
- 한국 연간 출생아 수 추이 및 전망
- 육아용품 시장 규모 (국내)
- 육아 앱/서비스 시장 규모
- 산후조리원 시장 규모 및 트렌드
- 육아 교육/콘텐츠 시장 규모

#### 1-3. 디지털 상품/크리에이터 이코노미
- 한국 디지털 상품 판매 플랫폼 현황 (크몽, 탈잉, 클래스101, 노션 템플릿 마켓 등)
- 육아 카테고리 디지털 상품 판매 현황 및 가격대
- 육아 분야 크리에이터/인플루언서 수익 모델
- 노션 템플릿, PDF 가이드 시장 트렌드

### 2. 경쟁 환경 분석

#### 2-1. 직접 경쟁 (아빠 육아 특화 서비스)
- 국내: 아빠 육아 전용 앱/서비스 존재 여부 및 현황
- 해외: DadApp, Fatherly, Dad's Playbook 등 해외 아빠 육아 서비스
- 각 서비스의 핵심 기능, 수익 모델, 사용자 수, 리뷰/평점

#### 2-2. 간접 경쟁 (범용 육아 서비스)
- 국내 주요 육아 앱: 베이비빌리, 그로우, 키즈노트, 맘맘 등
  - 핵심 기능, MAU, 수익 모델, 앱스토어 평점/리뷰
  - "아빠" 관련 기능 존재 여부
- 맘카페/커뮤니티: 네이버 맘카페, 블라인드 육아게시판
  - 아빠 관련 게시글 비중 및 주요 토픽
- 해외 주요 육아 앱: BabyTracker, Huckleberry, The Wonder Weeks

#### 2-3. 유사 바이럴 서비스 벤치마크
- "한국에가면" (whenivisitkorea) — 체크리스트 + 영수증 바이럴 구조 분석
- MBTI 테스트류 바이럴 앱 — 수익 모델, 트래픽 패턴, 리텐션
- Spotify Wrapped — 연말 바이럴 메커니즘
- 군 전역일 계산기 앱 — 특정 타겟 바이럴 사례

### 3. 콘텐츠/SNS 트렌드

#### 3-1. 아빠 육아 콘텐츠
- 인스타그램/스레드: #아빠육아, #대디 관련 해시태그 사용량 및 인기 콘텐츠 유형
- 유튜브: 아빠 육아 채널 (구독자 수, 조회수 상위 콘텐츠)
- 틱톡/릴스: 아빠 육아 숏폼 콘텐츠 트렌드
- 블로그: 아빠 육아 블로그 인기 주제, 검색 유입 키워드

#### 3-2. 바이럴 콘텐츠 패턴
- 최근 6개월 육아 관련 바이럴 콘텐츠 사례 수집
- 공유 트리거 분석 (공감, 유머, 충격, 유용성, 자랑)
- "남편에게 보내기" 류의 역방향 공유 패턴 사례
- 카카오톡 공유 vs 인스타 스토리 vs 스레드 — 채널별 확산 패턴

#### 3-3. 게이미피케이션 트렌드
- 비(非)게임 서비스의 게이미피케이션 성공 사례 (습관 앱, 학습 앱 등)
- XP/레벨/칭호 시스템 적용 사례 및 리텐션 효과
- 커플/가족 단위 게이미피케이션 사례

### 4. 기술/플랫폼 동향

#### 4-1. 기술 스택
- 바이럴 웹앱 최적 기술 스택 (Next.js, 이미지 생성, 공유 최적화)
- OG 태그 / 카카오톡 미리보기 최적화 방법
- 이미지 생성 (캔버스 API, html2canvas, 서버사이드 렌더링)
- 소셜 공유 API (카카오톡, 인스타 스토리, 스레드)

#### 4-2. 수익화 플랫폼
- 디지털 상품 판매: Gumroad, 크몽, 탈잉, 자체 결제
- 제휴 마케팅: 쿠팡파트너스, 아마존 어필리에이트
- 구독 결제: Stripe, 토스페이먼츠, 포트원
- 뉴스레터: 스티비, Mailchimp

## 리서치 방법론

### 데이터 소스

| 소스 유형 | 구체적 소스 | 수집 항목 |
|-----------|------------|-----------|
| 정부 통계 | 통계청, 여성가족부, 고용노동부 | 출생아 수, 육아휴직률, 가구 소비 |
| 검색 트렌드 | 네이버 데이터랩, Google Trends | 키워드 검색량 추이 |
| 앱 마켓 | App Store, Google Play | 육아 앱 순위, 리뷰, 다운로드 수 |
| SNS | 인스타그램, 스레드, 유튜브, 틱톡 | 해시태그 볼륨, 인기 콘텐츠, 크리에이터 |
| 커뮤니티 | 네이버 맘카페, 블라인드, 레딧 | 아빠 육아 관련 게시글, 반응, 주요 토픽 |
| 경쟁 서비스 | 각 서비스 웹사이트/앱 | 기능, UI, 수익 모델, 사용자 후기 |
| 리포트 | 닐슨, 오픈서베이, 한국콘텐츠진흥원 | 육아 시장 리포트, 디지털 소비 트렌드 |

### 분석 프레임워크

- **PEST 분석**: 정치(육아 정책), 경제(육아 비용/소비), 사회(아빠 역할 변화), 기술(앱/AI)
- **경쟁 포지셔닝 맵**: X축 = 정보성↔감성/재미, Y축 = 엄마 타겟↔아빠 타겟
- **바이럴 계수 분석**: 공유율, 전환율, 채널별 확산 속도 벤치마크

## 리서치 산출물

### 필수 산출물
1. **시장 트렌드 요약 보고서** — 핵심 수치 + 인사이트 (2~3페이지)
2. **경쟁 서비스 비교 매트릭스** — 기능/가격/타겟/수익모델 비교표
3. **바이럴 벤치마크 보고서** — 성공 사례 분석 + 파파레벨 적용 방안
4. **키워드/해시태그 분석** — 검색량 + SNS 볼륨 + 콘텐츠 갭
5. **기회 영역 도출** — 경쟁사가 못 하는 것, 시장이 원하는데 없는 것

### 선택 산출물
6. 지원금/정책 변화 타임라인 (연도별 정리)
7. 해외 아빠 육아 서비스 심층 분석
8. 기술 스택 비교표 (바이럴 웹앱 구현 옵션)

## 에이전트 실행 가이드

### 프롬프트 템플릿

```
당신은 "파파레벨(PapaLevel)" 프로젝트의 트렌드 리서치 에이전트입니다.

목표: 아빠 육아 시장의 트렌드, 경쟁 환경, 바이럴 패턴을 분석하여 
제품 의사결정에 필요한 인사이트를 도출합니다.

현재 리서치 주제: [주제 입력]

수행 지침:
1. 웹 검색을 통해 최신 데이터를 수집한다
2. 수치 데이터는 반드시 출처와 날짜를 명시한다
3. 경쟁 서비스는 직접 접속하여 현재 상태를 확인한다
4. 단순 나열이 아닌 "파파레벨에 주는 시사점"을 반드시 포함한다
5. 한국 시장을 우선으로 하되, 글로벌 확장 관점의 해외 사례도 포함한다

출력 형식:
- 핵심 발견사항 (3~5개 불릿)
- 상세 분석
- 파파레벨 시사점
- 추가 조사 필요 항목
```

### 실행 주기

| 주기 | 리서치 항목 |
|------|-----------|
| 1회성 (런칭 전) | 경쟁 분석, 시장 규모, 기술 스택 |
| 주간 | SNS 트렌드, 바이럴 콘텐츠 모니터링 |
| 월간 | 경쟁 서비스 업데이트, 신규 진입자 체크 |
| 분기 | 시장 트렌드 재평가, 정책 변화 반영 |
