# 유저 리서치 에이전트 (User Research Agent)

## 에이전트 목적

"파파레벨(PapaLevel)" 서비스의 핵심 타겟 사용자(아빠, 엄마)의 실제 니즈, 페인포인트, 행동 패턴, 구매 여정을 깊이 이해하여 제품 설계와 마케팅 전략의 근거를 마련한다.

---

## 타겟 사용자 정의

### Primary: 아빠 (직접 사용자)

#### 세그먼트 A — 예비 아빠 (출산 전)
- 아내 임신 중 (임신 초기~출산 직전)
- 핵심 상태: 불안 + 정보 부족 + "뭘 준비해야 하는지 모름"
- 정보 탐색 채널: 네이버 검색, 유튜브, 아내에게 전달받은 링크
- 소비 패턴: 아내가 리스트 주면 구매 실행, 본인 주도 구매는 카시트/유모차 정도

#### 세그먼트 B — 신생아 아빠 (0~6개월)
- 출산 직후, 산후조리원 또는 집에서 신생아 돌봄 중
- 핵심 상태: 수면 부족 + 멘탈 혼란 + "이게 맞나?" 불확실성
- 정보 탐색 채널: 새벽 수유 중 폰으로 검색, 블라인드, 동료 아빠에게 질문
- 소비 패턴: 급한 물건 즉시 구매 (쿠팡 새벽배송), 추가 육아용품

#### 세그먼트 C — 영아 아빠 (7~12개월)
- 이유식 시작, 아기 활동량 증가
- 핵심 상태: 약간 적응 + 피로 누적 + "나만 힘든가?" 고립감
- 정보 탐색 채널: 유튜브 이유식 레시피, 커뮤니티 후기
- 소비 패턴: 이유식 도구, 안전용품, 장난감

#### 세그먼트 D — 경험자 아빠 (둘째 이상)
- 이미 한 번 겪어본 아빠
- 핵심 상태: 여유 + "첫째 때 이걸 알았으면" 후회
- 정보 탐색 채널: 특정 필요할 때만 검색
- 소비 패턴: 선별적, 첫째 때 물건 재활용

### Secondary: 엄마 (바이럴 엔진 + 간접 사용자)

#### 세그먼트 E — 남편 육아 참여 유도형
- "남편이 좀 더 해줬으면" 하는 엄마
- 행동: 육아 정보/콘텐츠를 남편에게 카톡으로 전송
- 파파레벨 역할: "이거 해봐" → 남편 유입 트리거

#### 세그먼트 F — 육아 공유/자랑형
- 육아 일상을 SNS/맘카페에 적극 공유하는 엄마
- 행동: "우리 남편 전투력 C급ㅋㅋ" 같은 콘텐츠 공유
- 파파레벨 역할: 결과 카드를 2차 바이럴하는 확산자

## 리서치 항목

### 1. 니즈 & 페인포인트

#### 1-1. 출산 준비 단계 페인포인트
- 준비물 관련: "뭘 사야 하는지 모르겠다", "너무 많은 정보에 혼란", "뭐가 필수이고 뭐가 낭비인지"
- 비용 관련: "총 얼마나 드는지 감이 안 온다", "예상보다 많이 든다"
- 행정 관련: "출생신고 어떻게 하는지 모른다", "지원금 뭐가 있는지 모른다", "서류가 복잡하다"
- 심리적: "아빠가 될 준비가 된 건가", "실감이 안 난다"
- **검증 방법**: 네이버/블라인드에서 "출산 준비 남편", "예비 아빠 준비물" 관련 게시글 수집·분석

#### 1-2. 산후조리원 단계 페인포인트
- "면회 가서 뭘 해야 하는지 모르겠다"
- "아내 눈치 보인다", "장모님 앞에서 어색하다"
- "아기를 어떻게 안아야 하는지 모른다"
- "출생신고 등 행정을 혼자 처리해야 한다"
- **CK 직접 경험 기반 항목**: 빨대, 폰거치대, 멀티탭, 가습기 등 "아무도 안 알려주는 것들"
- **검증 방법**: "산후조리원 남편", "조리원 아빠" 키워드 커뮤니티 게시글 분석

#### 1-3. 퇴소 후 집 생활 페인포인트
- "새벽 수유가 이렇게 힘든 줄 몰랐다"
- "아내 산후우울인지 잘 모르겠다"
- "나도 힘든데 말할 데가 없다"
- "아기 울음을 멈출 수가 없다"
- **검증 방법**: "신생아 아빠 힘들다", "육아 아빠 멘탈" 관련 게시글 분석

#### 1-4. 엄마 측 페인포인트 (남편 관련)
- "남편이 도와준다고 하는데 뭘 해야 하는지 모른다"
- "시키면 하는데 알아서 하진 않는다"
- "남편이 체계적으로 알았으면 좋겠다"
- **검증 방법**: 맘카페에서 "남편 육아", "남편 도움" 관련 게시글 감성 분석

### 2. 행동 패턴

#### 2-1. 정보 탐색 행동
- 아빠들이 육아 정보를 어디서 찾는가? (검색 vs 커뮤니티 vs 지인 vs 아내)
- 탐색 시점은? (새벽 수유 중, 출퇴근 중, 주말)
- 어떤 형태의 정보를 선호하는가? (영상 vs 텍스트 vs 체크리스트 vs 인포그래픽)
- 정보 신뢰 기준은? (전문가 vs 경험자 vs 데이터)
- **검증 방법**: 네이버 검색 키워드 분석, 유튜브 아빠 육아 콘텐츠 댓글 분석

#### 2-2. 구매 행동
- 육아용품 구매 의사결정 프로세스 (누가 결정? 누가 실행?)
- 가격 민감도 (디지털 상품에 얼마까지 지불 의향?)
- 구매 채널 선호 (쿠팡 vs 네이버 쇼핑 vs 당근 vs 직접구매)
- **검증 방법**: 커뮤니티 구매 후기 분석, 쿠팡/네이버 쇼핑 리뷰 분석

#### 2-3. 소셜 공유 행동
- 아빠들이 육아 관련 콘텐츠를 공유하는가? 한다면 어디에?
- 어떤 콘텐츠를 공유하는가? (유머 vs 정보 vs 자랑 vs 공감)
- 엄마가 남편에게 보내는 콘텐츠 유형은?
- 역방향 공유 (엄마 → 아빠) 실제 발생 빈도와 전환률
- **검증 방법**: 인스타/스레드 #아빠육아 게시물 분석, 맘카페 "남편한테 보냈다" 류 게시글

#### 2-4. 앱/서비스 사용 행동
- 아빠들이 실제로 사용하는 육아 앱은? (앱 vs 그냥 검색?)
- 앱 설치 후 이탈 시점은? (온보딩? 1주? 1개월?)
- 유료 전환 트리거는? (기능 제한? 콘텐츠 품질? 커뮤니티?)
- **검증 방법**: 앱스토어 리뷰 분석 (특히 별 1~3개 리뷰에서 이탈 원인 추출)

### 3. 감정 여정 맵 (Emotional Journey Map)

#### 시기별 아빠의 감정 변화
```
임신 확인     임신 중기     출산 임박     출산 당일     조리원       퇴소 후      3개월차      6개월차
   |            |            |            |           |           |            |            |
  기쁨/        실감 안남/    불안/        감동/       어색함/     공포/        적응/        안정/
  설렘         무관심       조급함       혼란        눈치        수면부족     피로누적     자신감
```

- 각 시점에서 아빠가 가장 필요로 하는 것 (정보? 공감? 도구? 커뮤니티?)
- 각 시점에서 파파레벨이 제공할 수 있는 가치
- **검증 방법**: 시기별 커뮤니티 게시글 감성 분석, 아빠 육아 브이로그 코멘트 분석

### 4. 페르소나 구체화

#### 리서치 기반 페르소나 템플릿
```
이름: [가상]
나이: [20대 후반 ~ 40대 초반]
직업: [구체적]
아기 월령: [구체적]
거주지: [도시/지역]

하루 루틴:
- 06:00 [아기 활동]
- ...
- 24:00 [새벽 수유]

핵심 고통 3가지:
1. [구체적 상황]
2. [구체적 상황]  
3. [구체적 상황]

정보 탐색 패턴:
- 주로 [채널]에서 [시간대]에 검색
- 신뢰하는 정보원: [구체적]

파파레벨 사용 시나리오:
- 유입 경로: [아내 카톡? 블라인드? 검색?]
- 첫 사용 기능: [전투력 측정? 체크리스트?]
- 반복 사용 이유: [구체적]
- 유료 전환 트리거: [구체적]
```

### 5. 유료 전환 의향 조사

#### 5-1. WTP (Willingness to Pay) 리서치
- 체크리스트/가이드 PDF: 얼마까지 지불 의향? (0원 / 4,900원 / 9,900원 / 19,900원)
- 노션 템플릿: 얼마까지?
- 앱 월 구독: 얼마까지? (0원 / 1,900원 / 2,900원 / 4,900원)
- 어떤 기능/콘텐츠에 돈을 낼 의향이 있는가?
- **검증 방법**: 유사 디지털 상품 구매 후기에서 가격 언급 분석, 서베이(가능 시)

#### 5-2. 유료 전환 장벽
- "무료로 충분한 정보가 있는데 왜 사야 하나"
- "노션 템플릿이 뭔지 모른다"
- "결제가 번거롭다"
- 각 장벽별 해소 전략
- **검증 방법**: 디지털 상품 구매 거부 사유 수집 (리뷰, 커뮤니티)

## 핵심 리서치 질문 (Research Questions)

파파레벨 의사결정에 직접 영향을 주는 질문들:

### 제품 관련
1. 아빠들이 가장 먼저 해결하고 싶은 육아 문제 Top 3는 무엇인가?
2. 아빠들은 체크리스트형 도구를 선호하는가, 가이드형 콘텐츠를 선호하는가?
3. 게이미피케이션(XP, 레벨)이 아빠 육아에서 동기부여가 되는가, 오히려 부담인가?
4. "아내 인증" 시스템이 재미인가, 불쾌한가?
5. 커플 연동 기능이 실제 육아 참여를 높이는가?

### 바이럴 관련
6. 아빠들이 자발적으로 공유하는 콘텐츠 유형은 정확히 무엇인가?
7. 엄마 → 아빠 역방향 공유가 실제로 얼마나 자주 일어나는가?
8. "전투력 등급 카드"를 받으면 실제로 공유하겠는가?
9. 어떤 등급/결과가 가장 공유하고 싶은가? (높은 등급? 웃긴 등급?)
10. 카카오톡 공유 vs 인스타 스토리 — 어느 채널로 더 많이 공유하는가?

### 수익화 관련
11. 아빠들의 디지털 상품 구매 경험이 있는가? (노션 템플릿, 전자책 등)
12. 육아 관련 유료 콘텐츠에 지불할 의향이 있는 최대 금액은?
13. 구독 모델 vs 1회 구매 — 어느 쪽을 선호하는가?
14. 체크리스트에서 바로 구매 링크 연결 시 실제 클릭/구매 전환이 일어나는가?

### 리텐션 관련
15. 출산 후 육아 앱/서비스를 꾸준히 사용하는 기간은 얼마인가?
16. 아빠가 육아 서비스를 이탈하는 주요 원인은?
17. 월령이 올라가면서 새 콘텐츠가 필요한 시점은?
18. 커뮤니티(동기 아빠 그룹)가 리텐션에 실제로 효과가 있는가?

## 리서치 산출물

### 필수 산출물
1. **유저 페르소나 문서** — 세그먼트별 상세 페르소나 (최소 3개)
2. **페인포인트 우선순위 매트릭스** — 빈도 × 강도 × 해결 가능성
3. **감정 여정 맵** — 시기별 감정 + 니즈 + 터치포인트
4. **바이럴 행동 분석** — 공유 채널, 공유 동기, 공유 콘텐츠 유형
5. **WTP 분석** — 가격 수용도 + 유료 전환 장벽

### 선택 산출물
6. 커뮤니티 게시글 감성 분석 리포트
7. 경쟁 앱 UX 비교 분석
8. 마이크로 서베이 결과 리포트

| 우선순위 | 항목 | 이유 |

## 에이전트 실행 가이드

### 프롬프트 템플릿

```
당신은 "파파레벨(PapaLevel)" 프로젝트의 유저 리서치 에이전트입니다.

목표: 아빠 육아 서비스의 타겟 사용자(아빠/엄마)의 실제 니즈, 
페인포인트, 행동 패턴을 깊이 분석하여 제품 설계 근거를 마련합니다.

현재 리서치 주제: [주제 입력]

컨텍스트:
- CK는 2026년 2월생 아들의 아빠 (현재 1개월차)
- 실전 경험에서 나온 준비물 리스트가 있음 (빨대, 폰거치대, 멀티탭 등 현실 아이템)
- 핵심 가설: "엄마가 남편에게 보내는 행위"가 바이럴의 주요 엔진

수행 지침:
1. 추측이 아닌 실제 데이터/게시글/리뷰를 기반으로 분석한다
2. "아빠 관점"과 "엄마 관점"을 항상 분리하여 분석한다
3. 페인포인트는 구체적 상황("새벽 3시에 분유를 타는데...")으로 서술한다
4. 모든 인사이트에 "파파레벨에서 어떻게 해결할 수 있는가"를 포함한다
5. 가설은 가설로, 검증된 사실은 사실로 명확히 구분한다

출력 형식:
- 핵심 발견사항 (3~5개)
- 유저 인용/게시글 요약 (실제 목소리)
- 인사이트 → 제품 시사점
- 검증이 필요한 가설
```

### CK 일일 경험 기록 템플릿

```markdown
## 날짜: YYYY-MM-DD (아기 월령: X개월 Y일차)

### 오늘의 페인포인트
- 상황: [구체적으로]
- 감정: [솔직하게]
- 해결 방법: [어떻게 했는가]
- "이런 게 있었으면": [이상적 해결책]

### 오늘 구매/사용한 것
- 제품: [이름]
- 이유: [왜 필요했는가]
- 만족도: [1~5]
- 아빠 관점 리뷰: [한 줄]

### 아내와의 소통
- 갈등/조율 포인트: [있었다면]
- 아내가 보내준 링크/정보: [있었다면]
- 아내 컨디션: [관찰]

### 파파레벨 아이디어 메모
- [떠오른 기능/콘텐츠 아이디어]
```

### 실행 주기

| 주기 | 리서치 항목 |
|------|-----------|
| 매일 | CK 경험 기록 (5분) |
| 주 1회 | 커뮤니티 게시글 스캔 (30분) |
| 격주 1회 | 아빠 1명 비공식 인터뷰 (30분) |
| 월 1회 | 전체 인사이트 종합 → 제품 백로그 업데이트 |
| MVP 출시 후 | 사용자 행동 데이터 분석 + 피드백 수집 |

## 두 에이전트 연결 구조

```
트렌드 리서치 에이전트          유저 리서치 에이전트
        │                              │
   시장에 무엇이 있는가?         사용자가 무엇을 원하는가?
        │                              │
        └──────────┬───────────────────┘
                   │
                   ▼
            갭 분석 (Gap Analysis)
          "시장에는 없는데 사용자는 원하는 것"
                   │
                   ▼
            제품 의사결정
          기능 우선순위 / 가격 / 포지셔닝 / 바이럴 전략
                   │
                   ▼
              MVP 설계 & 실행
```

### 두 에이전트의 교차 질문

| 트렌드 에이전트가 발견 | 유저 에이전트가 검증 |
|----------------------|-------------------|
| "아빠 육아 검색량 증가 중" | 실제로 아빠가 검색하는 건가, 엄마가 대신 검색하는 건가? |
| "경쟁 앱 A가 커플 연동 출시" | 우리 타겟 아빠들은 커플 연동을 원하는가? |
| "게이미피케이션 트렌드 상승" | 육아에 게이미피케이션이 맞는가, 거부감이 있는가? |
| "노션 템플릿 시장 성장" | 아빠들이 노션을 쓰는가? 안 쓰면 대안은? |
| "숏폼 콘텐츠 트렌드" | 아빠들이 육아 숏폼을 보는 시간대/상황은? |
