## 1. 남편에게 보내기 (Send-to-Husband) 메시지

> 엄마가 카카오톡으로 남편에게 전송할 때 자동 삽입되는 메시지.
> 기본 포맷은 카카오톡 링크. 문체: 친근하고 가볍되 행동 유발이 명확하게.

### 1-1. 체크리스트 페이지 (`/checklist`)

```
여보, 이번 주에 이것만 해줘 👶
→ [링크]
```

**대안 A** (진행률 포함):
```
여보, 아직 이거 안 했잖아 👀
이번 주 아빠 할 일 확인해봐 → [링크]
```

**대안 B** (기한 임박 강조):
```
여보, 출생신고 {N}일 남았어.
지금 바로 확인해 → [링크]
```

### 1-2. 행정 타임라인 탭 (`/checklist?tab=admin`)

```
여보, 출생신고 {N}일 남았어! → [링크]
```

**대안** (첫만남이용권 강조):
```
여보, 200만 원짜리 신청 놓치면 안 돼.
{N}일 안에 해야 해 → [링크]
```

### 1-3. 예산 시뮬레이터 (`/budget`)

```
여보, 우리 아기 첫 해 얼마나 드는지 알아? → [링크]
```

**대안** (충격 숫자 포함):
```
여보, 산후조리원만 평균 287만 원이야.
전체 비용 한번 봐봐 → [링크]
```

### 1-4. 전투력 측정기 (`/quiz`)

```
여보, 이거 한번 해봐ㅋㅋ → [링크]
```

**대안** (도발형):
```
여보, 이 테스트 D등급 나오면 각오해ㅋㅋ → [링크]
```

## 2. 공유 메시지 (Share Messages)

> 아빠/엄마가 SNS·카톡에 직접 공유할 때 사용되는 메시지.
> 바이럴 공유(재미 중심)와 실행 공유(긴급성·실용 중심) 구분.

### 2-1. 전투력 결과 카드 (바이럴 공유)

**등급별 자동 삽입 메시지:**

| 등급 | 카톡 공유 문구 |
|------|--------------|
| S | 나 파파레벨 S등급 나왔는데 진짜임ㅋㅋ 아내도 믿는다고 함 → [링크] |
| A | 파파레벨 A등급. 장모님 공인 사위 레벨ㅋ → [링크] |
| B | 파파레벨 B등급... 의지는 S급이라고 합니다 → [링크] |
| C | 나도 루키 아빠래ㅋㅋ 너도 한번 해봐 → [링크] |
| D | 관전 아빠 D등급 나왔는데 ㄹㅇ 이거 누가 보내줬는지 생각해봐야 할 것 같음 → [링크] |

**예비 아빠 모드 (미출산) 공유 문구:**
```
나 곧 아빠 되는데 파파레벨로 준비 중ㅋ 같이 해봐 → [링크]
```

**OG 이미지 하단 워터마크 텍스트:**
```
papalevel.com | 아빠 되기 프로젝트, 여기서 관리하세요
```

### 2-2. 예산 영수증 카드 (바이럴 공유)

**기본 공유 문구:**
```
우리 아기 첫 해 예상 비용 {총액}원 나왔다.
지원금 빼도 {실질부담액}원이래... → [링크]
```

**대안 (충격 숫자 강조):**
```
아기 첫 해에 {총액}원 든대. 진짜임.
너도 계산해봐 → [링크]
```

**영수증 카드 상단 타이틀 (이미지 내 텍스트):**
```
🍼 우리 아기 첫 해 예상 비용
```

### 2-3. 이번 주 아빠 할 일 카드 (실행 공유)

**카드 내 고정 헤더:**
```
📋 이번 주 아빠 할 일 (D+{N})
```

**카드 공유 문구:**
```
여보, 이번 주 이것만 해줘 → [링크]
```

**카드 하단 고정 문구:**
```
파파레벨에서 더 보기 → papalevel.com
```

### 2-4. 행정 데드라인 카드 (실행 공유)

**카드 내 헤더:**
```
⚠️ 놓치면 안 되는 행정 마감
```

**카드 공유 문구:**
```
여보, {항목명} {N}일 남았어. 지금 해야 해 → [링크]
```

**예시 (출생신고):**
```
여보, 출생신고 9일 남았어. 지금 해야 해 → [링크]
```

**예시 (첫만남이용권):**
```
여보, 200만 원짜리 신청 53일 남았어. 잊지 마 → [링크]
```

## 3. CTA (Call-to-Action) 버튼 텍스트

> 맥락형 CTA: 페이지마다 다른 텍스트 사용. 일반형 "더 보기" 금지.

### 3-1. 전투력 측정기 결과 화면 → 타 기능 전환

| CTA 위치 | 버튼 텍스트 |
|----------|------------|
| 체크리스트로 (CTA 1) | 이번 주 아빠가 해야 할 3가지 → |
| 행정 타임라인으로 (CTA 2) | 행정 타임라인 바로 보기 → |
| 예산 시뮬레이터로 (CTA 3) | 첫 아이, 얼마나 들까요? → |

**예비 아빠 모드 CTA (미출산):**

| CTA 위치 | 버튼 텍스트 |
|----------|------------|
| 체크리스트로 | 출산 전 준비 체크리스트 보기 → |
| 행정으로 | 출생신고 전에 알아야 할 것들 → |
| 예산으로 | 얼마나 준비해야 할까? → |

### 3-2. 체크리스트 페이지 (`/checklist`)

| CTA 위치 | 버튼 텍스트 |
|----------|------------|
| 하단 고정 (남편에게 보내기) | 남편에게 보내기 |
| 행정 탭 진입 | 행정 타임라인 보기 |
| 예산으로 전환 | 이번 주 예산 확인하기 → |
| 항목별 구매 링크 | 쿠팡에서 보기 |
| 항목별 남편 공유 | 남편에게 보내기 |

### 3-3. 행정 타임라인 탭 (`/checklist?tab=admin`)

| CTA 위치 | 버튼 텍스트 |
|----------|------------|
| 하단 고정 | 행정 마감 남편에게 보내기 |
| 원스톱 신청 안내 | 정부24 바로가기 |
| 체크리스트로 전환 | 준비물 체크리스트 보기 → |

### 3-4. 예산 시뮬레이터 (`/budget`)

| CTA 위치 | 버튼 텍스트 |
|----------|------------|
| 진입 화면 헤드카피 | 산후조리원만 평균 287만 원. 첫 해 총 비용은? |
| 결과 화면 공유 | 예산 결과 카톡으로 공유 |
| 결과 화면 저장 | 예산 영수증 저장 |
| 체크리스트로 전환 | 준비물 체크리스트로 확인하기 → |
| 행정으로 전환 | 행정 타임라인 바로 보기 → |
| 하단 고정 | 남편에게 보내기 |

### 3-5. 랜딩 / 홈 (`/`)

| 위치 | 텍스트 |
|------|--------|
| 아빠 대상 태그라인 | 아빠 되기 프로젝트, 여기서 관리하세요 |
| 엄마 대상 태그라인 | 남편에게 이거 보내세요 |
| 주요 CTA (아빠 유입) | 전투력 측정하기 |
| 주요 CTA (엄마 유입) | 남편에게 보내기 |
| 체크리스트 진입 | 이번 주 아빠 할 일 확인하기 |
| 예산 진입 | 첫 해 비용 계산하기 |

### 3-6. 공유 fallback 시트 (Kakao SDK 실패 시)

| 단계 | 버튼 텍스트 |
|------|------------|
| 1순위 (카카오톡) | 카카오톡으로 보내기 |
| 2순위 (이미지 저장) | 이미지로 저장하기 |
| 3순위 (링크 복사) | 링크 복사 |
| iOS 안내 | 카카오톡 앱에서 직접 전달하거나 이미지를 저장해 보내세요 |
| Android 안내 | 카카오톡으로 바로 보내거나 이미지를 저장해 공유하세요 |

## 4. 정책 안내 문구 (Policy Notice)

> PRD v2 및 mvp-backlog 고정 원칙 그대로 반영.
> 모든 정책·지원금 항목에 아래 문구와 메타데이터를 함께 노출.

### 4-1. 지역별 차이 안내 (필수 — 모든 지원금 항목 상단)

**기본형:**
```
지역에 따라 달라질 수 있습니다.
아래 내용은 일반적인 기준이며, 거주지 기준으로 다시 확인하세요.
```

**카드/컴팩트형 (모바일 공간 제한 시):**
```
※ 지역·소득 기준에 따라 금액이 달라질 수 있습니다.
```

### 4-2. 기준일 / 최종 검수일 표기

**행정 타임라인 항목 하단:**
```
기준일: {YYYY-MM} | 최종 검수일: {YYYY-MM-DD}
```

**예시:**
```
기준일: 2025-01 | 최종 검수일: 2026-03-11
```

**예산 시뮬레이터 결과 화면 하단:**
```
지원금 기준: {YYYY}년 일반 기준 | 최종 검수일: {YYYY-MM-DD}
지역별 실제 지원금은 거주지 구청·복지로에서 확인하세요.
```

### 4-3. 개별 항목 주의 문구

**지자체 출산축하금 (지역 편차 큰 항목):**
```
지역마다 50만 원에서 200만 원 이상까지 크게 다릅니다.
본인 거주지 구청 홈페이지 또는 정부24에서 확인하세요.
```

**부모급여 (연도별 변동):**
```
2025년 기준 0세 월 100만 원, 1세 월 50만 원.
연도별 조정 가능성이 있으니 복지로에서 최신 기준을 확인하세요.
```

**산후도우미 지원사업 (소득·지역 기준):**
```
소득 기준과 지역에 따라 지원 금액이 다릅니다.
신청 가능 기간: 출산예정일 40일 전 ~ 출산 후 30일.
복지로 또는 거주지 보건소에서 신청하세요.
```

**육아휴직 급여 (제도 변동 가능):**
```
6+6 부모육아휴직제 적용 시 급여가 달라집니다.
최신 기준은 고용보험 홈페이지(ei.go.kr)에서 확인하세요.
```

### 4-4. 전체 예산 결과 화면 법적 고지 (하단 소형 텍스트)

```
이 시뮬레이터는 참고용이며, 실제 지원 금액 및 조건은 관련 기관에서 반드시 확인하세요.
정책 정보 기준일: {YYYY-MM} | 최종 검수일: {YYYY-MM-DD}
```

## 6. 리스크

| 리스크 | 내용 | 권장 대응 |
|--------|------|----------|
| 카카오톡 메시지 길이 제한 | 카카오 링크 메시지는 200자 이내 권장. 현재 초안 대부분 100자 이하로 안전 범위. | 최종 SDK 연동 시 실 기기 테스트 필수 |
| 등급별 공유 문구 바이럴 효과 검증 미완 | D등급 문구가 가장 바이럴 가능성 높으나 미검증 | P0: 추가 검증 과제 (인터뷰 또는 시안 테스트) |
| 정책 정보 최신성 | 부모급여·지자체 축하금 등 연도별 변동 가능 | 기준일/검수일 메타 필드 필수. 분기별 검수 일정 수립 |
| 지역 안내문구 모바일 과밀 | 카드 UI에서 안내문구가 핵심 정보를 가릴 수 있음 | 컴팩트형(※ 단행 버전) 사용 + 상세 안내는 토글/하단 배치 |
| 미출산 예비 아빠 모드 카피 분기 | PRD에서 미출산 모드 CTA 고정 지시됨. 낮은 등급 낙인 카피 사용 금지 | 예비 아빠 모드 분기 조건 구현 시 이 카피 파일과 동기화 필수 |
