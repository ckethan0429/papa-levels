| 배지 | 의미 |
|------|------|
| `필수` | 반드시 해야 함 |
| `권장` | 강력 권장 |
| `선택` | 여유 있으면 |

## 주요 설계 결정 (upstream lock 반영)

- `행정 타임라인`은 이 화면 내 4번째 탭(`?tab=admin`)으로 운영. 독립 라우트 없음.
- `남편에게 보내기` 하단 고정 CTA는 모든 탭에서 유지.
- 공유 기본 포맷: 카카오톡 링크 → 이미지 저장 → 링크 복사 순서.
- 사회적 증거(`아빠 N%가 체크` 등) MVP 미노출.
- 로그인 없음. localStorage + 익명 집계.
- Context Summary Bar의 입력값은 `/budget`, `/quiz`와 공용.

## 리스크

- `남편에게 보내기` 하단 고정 CTA가 항목 체크 영역 또는 탭 전환 버튼을 가릴 수 있음 → 스크롤 시 자동 숨김 또는 미니화 검토 필요.
- 항목 카드 펼침/접힘 UX에서 지역 안내문구 + 기준일 메타데이터가 모바일 화면에서 과밀해질 수 있음.
