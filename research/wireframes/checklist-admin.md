## 딥링크 진입 규칙

| 진입 경로 | 동작 |
|-----------|------|
| `/checklist?tab=admin` 직접 URL | admin 탭 활성, Context Summary Bar 공용 입력값 유지 |
| 다른 탭 → 행정 탭 클릭 | 탭 전환, 스크롤 상단 이동 |
| 공유 링크 수신 → 진입 | admin 탭 활성, 로컬 저장 상태 없으면 온보딩 입력 후 복귀 |
| 결과 화면 CTA "행정 타임라인 바로 보기 →" | admin 탭으로 딥링크 |

## 주요 설계 결정 (upstream lock 반영)

- 독립 라우트(`/admin`) 없음. `/checklist?tab=admin` 고정.
- 원스톱 추천 순서 + 마감 임박 카드를 최상단 배치.
- 정책/지원금 항목은 `지역에 따라 달라질 수 있습니다` + `기준일` + `최종 검수일` 필수 노출.
- 사회적 증거 미노출.
- 체크 상태는 `/checklist` 탭과 동일한 localStorage 저장.
