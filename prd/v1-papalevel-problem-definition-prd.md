# Part 1. 문제정의 (Problem Definition)

---

## 1. 문제 한 줄 요약

> **한국 아빠들은 출산 전후에 "무엇을, 언제, 어떻게 해야 하는지" 모르고, 그 정보를 찾을 곳도, 물어볼 곳도 없다.**

---

## 2. 문제의 맥락

### 2-1. 시장 상황

한국의 남성 육아휴직 사용률은 빠르게 증가하고 있다. 2019년 약 7%에서 2024년 약 30%까지 올라왔으며, 정부는 6+6 부모육아휴직제, 부모급여 확대 등 아빠 육아 참여를 적극 유도하고 있다. 사회적으로 "아빠도 육아한다"는 인식이 보편화되고 있다.

그러나 **인식의 변화에 비해 인프라(정보, 도구, 커뮤니티)는 거의 없다.** 육아 정보 시장은 99%가 엄마 중심이다. 맘카페, 육아 앱, 육아 블로그, 육아 유튜브 — 모두 엄마를 위해 엄마가 만든 콘텐츠다.

### 2-2. 아빠가 처한 현실

아빠의 출산/육아 여정을 시간순으로 보면:

**임신 중**: 아내가 정보를 수집하고, 아빠는 "시키는 것"을 한다. 자발적으로 준비하고 싶어도 어디서부터 시작할지 모른다. "출산 준비물"을 검색하면 엄마 관점 리스트만 나온다. 아빠 전용 체크리스트는 존재하지 않는다.

**출산 직후 (산후조리원)**: 가장 혼란스러운 시기다. 분만실에서 뭘 해야 하는지, 산후조리원 면회 때 뭘 가져가야 하는지, 출생신고는 언제 어떻게 하는지, 지원금은 뭐가 있는지 — 이 모든 정보가 흩어져 있고, 아무도 체계적으로 정리해주지 않는다. "빨대"가 필요하다는 것, "멀티탭"이 필수라는 것은 겪어봐야 안다.

**퇴소 후 (집)**: 새벽 수유, 기저귀 교체, 아기 목욕 — 이론으로는 알지만 실전은 다르다. 아내는 산후 회복 중이라 아빠가 해야 할 일이 많은데, 순서와 방법을 가르쳐주는 곳이 없다. 아빠의 멘탈 관리는 아예 의제에도 오르지 않는다.

### 2-3. 기존 솔루션의 한계

| 기존 솔루션 | 한계 |
|------------|------|
| 네이버 검색 | 엄마 관점 블로그만 나옴, 정보가 오래됨, 체계적이지 않음 |
| 맘카페 | 아빠가 질문하기 어려운 분위기, 엄마 중심 커뮤니티 |
| 육아 앱 (베이비빌리, 그로우 등) | 아기 기록 중심, 아빠 온보딩/가이드 부재 |
| 유튜브 | 검색해야 나옴, 체계적이지 않음, 시기별 맞춤 정보 없음 |
| 책/가이드북 | 범용적, 한국 행정/지원금 정보 없음, 시의성 떨어짐 |
| 주변 지인 | 운이 좋아야 경험 있는 아빠가 있음, 체계적이지 않음 |

### 2-4. 핵심 문제 구조화

```
[근본 원인]
육아 정보 생태계가 엄마 중심으로 구축됨
  │
[직접 원인]
아빠 전용 정보/도구/커뮤니티 부재
  │
[증상 — 아빠 측]
├─ 뭘 준비해야 하는지 모름 (정보 부재)
├─ 총 비용이 얼마인지 모름 (재무 불확실성)
├─ 행정 처리 순서를 모름 (행정 복잡성)
├─ 실전 노하우가 없음 (경험 부재)
├─ 물어볼 곳이 없음 (커뮤니티 부재)
└─ 자신의 역할이 불확실함 (역할 모호성)
  │
[증상 — 엄마 측]
├─ "남편이 뭘 해야 하는지 모른다" (기대 vs 현실 갭)
├─ 일일이 시켜야 한다 (정신적 부담)
└─ 남편에게 체계적으로 알려줄 수단이 없다 (도구 부재)
```

## 3. 우리의 관점 (Point of View)

### 아빠에게
> 출산은 프로젝트다. 모든 프로젝트에는 체크리스트, 일정, 예산, 실행 매뉴얼이 있다.
> 그런데 "아빠 되기" 프로젝트에는 아무것도 없다.
> 파파레벨은 이 프로젝트의 PM 도구를 제공한다.

### 엄마에게
> "남편에게 일일이 말 안 해도, 이거 보내주면 된다."

### 사회에
> 아빠의 육아 참여는 의지의 문제가 아니라 인프라의 문제다.
> 정보와 도구가 있으면 아빠도 한다.

## 4. 타겟 사용자

### Primary User: 아빠
- 1차 타겟: 첫째 아이, 출산 전 3개월 ~ 출산 후 6개월
- 연령: 28~40세 (평균 초산 연령 기준)
- 특성: 직장인, 육아 정보 탐색 의지는 있으나 시작점을 모름

### Secondary User: 엄마 (바이럴 엔진)
- 남편에게 정보를 전달하는 역할
- "이거 해봐" → 카카오톡 공유 → 아빠 유입

### Anti-Persona (타겟이 아닌 사용자)
- 육아에 관심 없는 아빠 (도구를 줘도 안 쓸 사람)
- 이미 다양한 육아 앱을 사용하는 적극적 아빠 (기존 도구로 충분한 사람)

## 2. MVP 기능 명세

### Feature 1: 아빠 육아 전투력 측정기

#### 개요
5개 질문에 답하면 아빠의 "육아 전투력 등급"을 진단하고, SNS 공유용 결과 카드를 생성한다.

#### 핵심 목적
- 바이럴 유입 엔진 (공유를 통한 신규 사용자 획득)
- 재미 + 공감을 통한 첫 인상 형성
- 다른 기능(예산 시뮬레이터, 체크리스트)으로의 자연스러운 전환

#### 사용자 플로우

```
[랜딩] "당신의 육아 전투력은?"
  │
  ▼
[질문 1/5] 객관식 선택 (탭/클릭)
  │
  ▼
[질문 2/5] ...
  │
  ▼
[질문 5/5] ...
  │
  ▼
[결과 화면]
  ├─ 등급 카드 (이미지)
  ├─ 등급 설명 + 재치 있는 코멘트
  ├─ "전체 아빠 중 상위 X%" 표시
  ├─ [카카오톡 공유] [인스타 스토리 공유] [이미지 저장]
  └─ [CTA] "예산도 체크해볼까요?" / "준비물 확인하기"
```

#### 질문 설계

| # | 질문 | 보기 | 배점 기준 |
|---|------|------|----------|
| 1 | 아기 울음소리에 몇 초 만에 눈을 뜨나요? | 3초 이내 / 10초 / 아내가 깨워줌 / 모름(아직 미출산) | 경험도 + 반응속도 |
| 2 | 기저귀 교체, 자신 있나요? | 눈 감고도 가능 / 해봤다 / 이론만 / 기저귀 종류를 모름 | 실전 경험 |
| 3 | 분유 적정 온도를 어떻게 확인하나요? | 손목 테스트 / 입으로 / 온도계 / 분유를 타본 적 없음 | 실무 지식 |
| 4 | 아기와 단둘이 외출한 최장 시간은? | 3시간+ / 1시간 / 편의점까지 / 아직 없음(미출산 포함) | 독립 돌봄 능력 |
| 5 | 아기 재우기, 본인만의 비법이 있나요? | 3가지 이상 / 하나 있음 / 매번 다름 / 재운 적 없음 | 노하우 축적도 |

- 미출산 아빠도 참여 가능하도록 각 질문에 "아직 미출산" 보기 포함
- 점수 합산 → 등급 산출

#### 등급 체계

| 등급 | 점수 범위 | 칭호 | 결과 코멘트 |
|------|----------|------|------------|
| S | 23~25 | 전설의 슈퍼대디 | "아내가 이 결과를 보면 의심할 수 있습니다" |
| A | 18~22 | 숙련된 육아전사 | "장모님이 인정한 사위 레벨입니다" |
| B | 13~17 | 성장형 아빠 | "의지는 S급, 스킬은 업데이트 중" |
| C | 8~12 | 루키 아빠 | "모든 레전드도 여기서 시작했습니다" |
| D | 5~7 | 관전 아빠 | "지금 이 테스트를 누가 보내줬는지 생각해보세요" |

- 모든 등급의 코멘트가 재미있어야 공유된다 (낮은 등급일수록 더 웃기게)
- D등급 멘트가 가장 바이럴 확률 높음

#### 결과 카드 사양

```
크기: 1080 x 1080px (인스타 정사각) + 1080 x 1920px (스토리/릴스)
구성 요소:
  - 등급 아이콘 (S/A/B/C/D)
  - 칭호 텍스트
  - "전체 아빠 중 상위 X%"
  - 레이더 차트 (5개 질문 영역별 점수 시각화)
  - papalevel.com 워터마크
  - QR 코드 (스캔 → 측정 페이지)
디자인 톤: 게임 캐릭터 카드 스타일, 밝고 유쾌한 컬러
```

#### 기술 요구사항
- 이미지 생성: html2canvas 또는 서버사이드 렌더링
- 카카오톡 공유: Kakao SDK (링크 공유 + 커스텀 OG 이미지)
- 인스타 스토리 공유: 이미지 다운로드 → 스토리 공유 유도
- 결과 데이터 저장: 등급 분포 통계용 (익명, 집계 목적)

### Feature 2: 출산 예산 시뮬레이터

#### 개요
출산 전후 비용 항목을 체크하면 총 예상 비용을 산출하고, 정부 지원금 차감 후 실질 부담액을 보여준다.

#### 핵심 목적
- 실용적 가치 제공 (바이럴 유입 후 잔류 이유)
- "충격 숫자" 바이럴 (총 비용 SNS 공유)
- 디지털 상품(서바이벌 키트)으로의 자연스러운 전환
- 제휴 마케팅 연결 (항목별 구매 링크)

#### 사용자 플로우

```
[시작] "첫 아이, 얼마나 들까요?"
  │
  ▼
[기본 정보 입력]
  - 출산 예정/출산 완료 선택
  - 거주 지역 (지자체별 지원금 차이)
  - 맞벌이 여부 (육아휴직 급여 계산)
  │
  ▼
[카테고리별 항목 체크 + 등급 선택]
  - 출산/의료
  - 산후조리
  - 육아용품 필수
  - 육아용품 선택
  - 월간 소모품
  - 숨은 비용
  │
  ▼
[결과 화면]
  ├─ 총 예상 비용 (첫 해 기준)
  ├─ 정부 지원금 총액
  ├─ 실질 부담액 (= 총 비용 - 지원금)
  ├─ "같은 지역 아빠 평균과 비교" (사회적 증거)
  ├─ 항목별 절약 팁
  ├─ [카카오톡 공유] [이미지 저장]
  └─ [CTA] "준비물 체크리스트로 확인하기" / "절약 가이드 받기"
```

### Feature 3: 출산 준비물 체크리스트

#### 개요
출산 전후 필요한 모든 준비물을 시기별·카테고리별로 정리하고, 아빠 관점의 실전 코멘트를 제공한다. "다른 아빠 체크율"로 사회적 증거를 보여준다.

#### 핵심 목적
- 실용적 핵심 가치 (가장 높은 리텐션 기능)
- 사회적 증거 ("아빠 91%가 필수라고 체크")
- 제휴 마케팅 수익화 (항목별 구매 링크)
- 디지털 상품 업셀 (상세 가이드)

#### 사용자 플로우

```
[시작] 시기 선택
  ├─ 출산 전 준비 (D-30 ~ D-Day)
  ├─ 산후조리원 (D-Day ~ D+14)
  ├─ 퇴소 후 집 세팅 (D+14 ~ D+30)
  └─ 지원금/행정
  │
  ▼
[카테고리 화면] 
  - 카테고리별 탭
  - 각 항목: 체크박스 + 이름 + 아빠 코멘트 + "아빠 N%가 체크"
  - 필수/권장/선택 라벨
  │
  ▼
[항목 상세] (항목 클릭 시)
  - 왜 필요한가 (아빠 관점 실전 설명)
  - 예산 범위 (절약/표준/프리미엄)
  - 중고 가능 여부 + 중고 구매 팁
  - 아빠 실전 팁 (CK 직접 경험)
  - [구매 링크] (쿠팡파트너스)
  │
  ▼
[요약 화면]
  - 체크한 항목 수 / 전체 항목 수
  - 카테고리별 완료율
  - 예산 시뮬레이터 연동 (체크한 항목 기반 총액)
  - [체크리스트 이미지 저장] [카카오톡 공유]
```

### 4-1. 성능
- First Contentful Paint: < 1.5초
- 전투력 측정 완료 ~ 결과 카드 생성: < 2초
- 이미지 생성: < 3초
- Lighthouse 점수: 90+ (모바일)

### 4-2. 반응형
- Mobile First (주 사용 환경: 카톡에서 클릭 → 모바일 브라우저)
- 지원 해상도: 360px ~ 428px (주력), 태블릿/데스크톱 호환
- 카카오 인앱 브라우저 호환 필수

### 4-3. SEO / OG
- 각 기능별 개별 URL (전투력: /quiz, 예산: /budget, 체크리스트: /checklist)
- OG 이미지: 기능별 커스텀 (전투력 결과 카드 = OG 이미지)
- 네이버 검색 최적화: "아빠 출산 준비물", "출산 비용 계산기" 키워드

### 4-4. 분석
- Google Analytics 4 (페이지뷰, 이벤트)
- 핵심 이벤트 트래킹:
  - quiz_start, quiz_complete, quiz_share
  - budget_start, budget_complete, budget_share
  - checklist_check, checklist_share
  - cta_click (기능 간 전환)
  - affiliate_click (제휴 링크 클릭)

## 5. 기술 스택 (제안)

| 영역 | 기술 | 이유 |
|------|------|------|

| 분석 | GA4 + Mixpanel (무료 티어) | 이벤트 트래킹 |
| 공유 | Kakao SDK | 카카오톡 공유 필수 |
| 제휴 | 쿠팡파트너스 API | 구매 링크 + 수익 |

## 8. 리스크 및 대응

| 리스크 | 영향 | 확률 | 대응 |
|--------|------|------|------|
| 아빠들이 공유하지 않음 | 바이럴 실패 | 중간 | 엄마→아빠 역방향 공유 경로 강화 |
| 유사 서비스 등장 | 차별화 약화 | 낮음 | CK 실전 경험 기반 콘텐츠로 차별화 (모방 불가) |
| 정부 지원금 정책 변경 | 시뮬레이터 정확도 하락 | 높음 | 분기별 업데이트 + "2026년 기준" 명시 |
| 쿠팡파트너스 수수료율 변경 | 제휴 수익 감소 | 중간 | 다채널 제휴 (네이버쇼핑, 직접 제휴) |
| 출산율 지속 하락 | 시장 축소 | 높음 | 질적 성장(객단가 향상) + 글로벌 확장(PapaLevel 영어 버전) |
